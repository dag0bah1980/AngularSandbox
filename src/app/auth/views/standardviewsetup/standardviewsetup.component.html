This page describes how to set up a standard view component. This has been updated as of 03/16/2019.

For a view component, the type script needs to have the following includes:

import {{ '{' }} ViewMethodsService {{ '}' }} from '../../services/sharedMethods/view-methods.service';
import {{ '{' }} ActivatedRoute, Router, {{ '}' }} from '@angular/router';

As well, the constructor and the onInit need to look like this:

constructor(private _viewMethodsService: ViewMethodsService, public _activatedRoute: ActivatedRoute) {{ '{' }}
this._viewMethodsService.updateTitle(this._activatedRoute);
{{ '}' }}
ngOnInit() {{ '{' }}
this._viewMethodsService.updateTitle(this._activatedRoute);
{{ '}' }}

Finally, if you have a view component that has child components, the HTML needs to include the following:

<div *ngIf="_activatedRoute.children.length === 0">
<p>
blah works!
</p>
</div>
<router-outlet></router-outlet>
